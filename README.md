# Smart E-commerce Application

This is a microservices-based Smart E-commerce application developed using **Python** and **Node.js**.  
It includes multiple services for a complete e-commerce system:

## Features
- **ATM Management System**: Deposit, withdraw, transfer, and account management
- **Cart Service**: Add/remove products to cart
- **Order Service**: Create and manage orders
- **Product Service**: Manage product catalog
- **User Service**: User registration, login, and authentication
- **Notification Service**: Email and notification system

## Tech Stack
- Backend: Python (Flask), Node.js
- Database: SQLite, MongoDB
- Microservices architecture
- Docker & Docker Compose for containerization

## How to Run
1. Clone the repository
2. Install dependencies using `pip` or `npm`
3. Run services individually or using `docker-compose up`
4. Open browser at `http://localhost:5000` (for ATM system)
